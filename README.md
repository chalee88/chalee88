## 🔬 Research Interests

**Brain–Computer Interfaces (BCI)**
Developing systems that establish direct communication pathways between the brain and external devices. My focus is on non-invasive EEG-based BCIs — particularly motor imagery decoding, where users can control devices through imagined movement without any physical action.

**Neural Signal Decoding**
Extracting meaningful information from raw neural signals using both classical approaches (CSP, LDA) and modern deep learning architectures (EEGNet, CNN-LSTM, GCN). I'm especially interested in making decoders that generalise across subjects, sessions, and conditions.

**Signal Processing for Physiological Data**
Applying time-frequency analysis, spatial filtering, and adaptive filtering techniques to ECG, EEG, and PCG signals. This includes both analog hardware design (instrumentation amplifiers, active filters) and software pipelines using MNE and MATLAB.

**Machine Learning for Physiological Signals**
Designing and benchmarking deep learning architectures — CNNs, LSTMs, GCNs, and hybrid models — for clinical tasks like arrhythmia detection, seizure classification, and motor imagery decoding. I pay close attention to class imbalance, cross-subject generalisation, and reproducibility.

**Assistive Neurotechnology**
Using BCI and embedded systems to build tools that restore or augment human capability — from posture monitoring cushions to future work in neuroprosthetics and assistive communication devices.
